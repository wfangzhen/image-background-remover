# 图片背景移除工具 - MVP 需求文档

## 1. 项目概述

| 项 | 内容 |
|---|---|
| **项目名称** | image-bg-remover |
| **核心功能** | 上传图片，自动移除背景，返回透明PNG |
| **目标用户** | 设计师、电商卖家、内容创作者 |
| **上线周期** | 1-2天 |

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽 + 点击上传，格式：JPG/PNG/WebP，最大 10MB | P0 |
| 背景移除 | 调用 Remove.bg API 移除背景 | P0 |
| 结果预览 | 实时显示原图 vs 处理后对比 | P0 |
| 图片下载 | 一键下载透明PNG | P0 |
| 进度提示 | 处理中显示加载动画 | P0 |

### 2.2 边界情况

- 文件超过 10MB → 提示"文件过大"
- 格式不支持 → 提示"仅支持 JPG/PNG/WebP"
- API 调用失败 → 提示"处理失败，请重试"
- API 额度用完 → 提示"今日额度已用完"

---

## 3. 技术方案

### 3.1 架构

```
[用户浏览器]
    ↓ HTTP POST (图片base64)
[Cloudflare Worker] → 环境变量读取 REMOVE_BG_API_KEY
    ↓ 转发
[Remove.bg API]
    ↓ 返回透明PNG
[用户浏览器] → 展示 & 下载
```

### 3.2 存储

- **零存储**：图片全程内存处理，不落盘
- Worker 无状态，可无限横向扩展

### 3.3 安全

| 安全项 | 方案 |
|--------|------|
| API Key 保护 | 仅存在于 Worker 环境变量，不暴露客户端 |
| 请求校验 | Worker 限制仅允许 POST + JSON body |
| CORS | 配置允许的域名 |
